# TinyChat
**基于socket实现的多人在线聊天室**

本项目前端采用Qt设计可视化界面，完成上线，下线，发送/接收消息，日志提醒，好友列表选择等功能，后端部署在linux上,使用c++基于socket开发，实现了消息的转发，可支持一定的并发性。
